명명 / Naming

컴포넌트는 PascalCase, 폴더와 파일은 kebab-case, 공개 CSS class는 ds- prefix를 사용합니다. / Components use PascalCase, folders and files use kebab-case, and public CSS classes use the ds- prefix.

상태 / State

동적 스타일은 data-state, data-disabled, data-invalid, data-orientation 같은 속성으로 표현합니다. / Dynamic styles use attributes such as data-state, data-disabled, data-invalid, and data-orientation.

토큰 / Tokens

컴포넌트는 원시 색상값 대신 --ds-color-*, --ds-space-*, --ds-radius-* 토큰을 참조합니다. / Components reference --ds-color-*, --ds-space-*, and --ds-radius-* tokens instead of raw style values.

접근성 / Accessibility

네이티브 HTML을 먼저 사용하고, 커스텀 상호작용은 WAI-ARIA APG 패턴을 기준으로 정의합니다. / Use native HTML first, and define custom interaction with WAI-ARIA APG patterns.
