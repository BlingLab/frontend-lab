Conventions

개발 규약

명명

컴포넌트는 PascalCase, 폴더와 파일은 kebab-case, 공개 CSS class는 ds- prefix를 사용합니다.

상태

동적 스타일은 data-state, data-disabled, data-invalid, data-orientation 같은 속성으로 표현합니다.

토큰

컴포넌트는 원시 색상값 대신 --ds-color-*, --ds-space-*, --ds-radius-* 토큰을 참조합니다.

접근성

네이티브 HTML을 먼저 사용하고, 커스텀 상호작용은 WAI-ARIA APG 패턴을 기준으로 정의합니다.
